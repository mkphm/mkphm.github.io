Coursework

Fundamentals of Web Development

PROG 109

Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects. Prerequisite: Placement by assessment into ENGL& 101, or completion of ENGL 092 or 093 with a C or better.

Class uses HTML, CSS and JavaScript.

Instructor: Adam Brehm

Reflection

Before taking the class, I have no prior knowledge of any programming languages. But I have always have an interested in it because I like building things and making them look good.

Now after taking this class, I can use HTML, CSS and some JavaScript. I am able to build a simple website from scratch and implement some functions into that website.

I used get frustrated when a minor inconvenience occurs while I browse a website. But now that I have an understanding of what goes behind making a website, I have a great appreciation for the craft. I think I am more forgiving when a developer made a mistakes when building the website now.
